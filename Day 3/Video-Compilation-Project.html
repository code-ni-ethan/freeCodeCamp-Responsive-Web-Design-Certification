Why I Love and Hate Programming

Like all other programmers, I have a love-hate relationship with programming. It's like human relationships, some days you get along so well, and suddenly you're both at war because you cannot understand each other! But in an objective sense, I do have love for this topic because programming consists of critical thinking, decision-making skills, and trains me to dissect real world problems into smaller ones for easier completion.

My Favorite Languages

C++

My best language where I excel the most. C++ is a high-level, general-purpose programming language created by Danish computer scientist Bjarne Stroustrup.

Java

Probably one that I need to relearn. Its syntax is quite challenging for me, but I'll probably get used to it when I get to learn it all over again real soon. Java is a high-level, general-purpose, memory-safe, object-oriented programming language. It is intended to let programmers write once, run anywhere (WORA), meaning that compiled Java code can run on all platforms that support it.

MySQL

Super beginner at this one. Haven't even touched a single minute of this course yet, but I'm quite excited to learn it together with Java. MySQL is an open source relational database management system (RDBMS) that’s used to store and manage data. Its reliability, performance, scalability, and ease of use make MySQL a popular choice for developers. In fact, you’ll find it at the heart of demanding, high-traffic applications such as Facebook, Netflix, Uber, Airbnb, Shopify, and Booking.com.
